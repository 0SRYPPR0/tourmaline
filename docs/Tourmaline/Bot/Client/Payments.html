    def answer_pre_checkout_query(pre_checkout_query_id, ok, error_message) #

    Once the user has confirmed their payment and shipping details, the Bot API sends the final confirmation in the form of a Model::Update with the field pre_checkout_query. Use this method to respond to such pre-checkout queries. On success, true is returned.

    Note: The Bot API must receive an answer within 10 seconds after the > pre-checkout query was sent.
