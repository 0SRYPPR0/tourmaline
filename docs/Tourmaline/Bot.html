    module Tourmaline::Bot

    Overview

    The Bot module is an interface to Telegram's bot API. With it you can create bots to use on the telegram platform, which can do everything from providing useful (or useless) interactions, to moderating chats.

    To create a bot, see the examples in the Bot::Client class.
